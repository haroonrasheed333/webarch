About the Project

This project is our mobile version of bitly.

The user enters Long URL for which short URL is generated. The user can also specify a specific short path for their long URL.

The app also shows the following analysis.

  • Most Followed URLs
  • Trending URLs and
  • Counts/Action per Browser
